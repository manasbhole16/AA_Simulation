Kruskal's Algorithm Visualization

How to use:

  1. Enter the number of nodes in your graph
  2. Add edges by specifying source node, destination node, and weight
  3. Click "Visualize" to see Kruskal's algorithm in action
  4. Use Step-by-Step mode to see each decision the algorithm makes

Example: A graph with 5 nodes (0-4) might have edges like: (0,1,2), (0,3,6), (1,2,3), (1,3,8), (1,4,5), (2,4,7), (3,4,9)
